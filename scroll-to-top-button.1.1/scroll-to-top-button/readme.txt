Displays a Red button in accordance with UW theme color in the bottom right corner for smooth scrolling to the top of the page.

== Description ==

A simple WordPress plugin which adds a "scroll to top" button to the bottom corner of the page if the page has been scrolled down. It is compatible with all major browsers and should work with all themes, including
responsive and high resolution themes.

Plugin has been manually set to red and no longer contains dark or light color schemes.

If your theme doesn't already contain this feature, you can simply add it by means of this plugin.

== Installation ==

1. Download the latest version and extract it in the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

For uninstallation, deactivate the plugin in the 'Plugins' menu in WordPress.
